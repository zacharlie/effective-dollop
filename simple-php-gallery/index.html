Gallery

This gallery is a simple HTML template for creating a pretty list of items, information and links with CSS "cards".
You could use it on a storefront, landing page, dashboard or wherever you like.
It uses the bootstrap 4 CSS framework for responsive design and Font Awesome icons for inclusion of element symbols.
A very simple example of using this with a csv list and pagination using php is available in the effective-dollop repo.
